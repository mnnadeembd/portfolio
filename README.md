#  Personal Portfolio Website

A modern, responsive, and animated personal portfolio built to showcase my skills, projects, and experience as a Full-Stack Web Developer.

---

##  Live Demo
 [View Live Site](http://nuruzzaman.intelsofts.com)

---

##  Features

-  Modern UI/UX design
-  Smooth animations (AOS + custom effects)
-  Fully responsive (mobile-first)
-  Scroll to top & scroll indicators
-  Interactive hero section with particle background
-  Contact form with backend email auto-reply
-  Clean and reusable component structure

---

##  Tech Stack

### Frontend
- React.js
- JavaScript (ES6+)
- Bootstrap /  CSS
- AOS Animation
- tsparticles

### Backend
- Laravel
- PHP
- MySQL
- Mail system (SMTP for auto-reply)
